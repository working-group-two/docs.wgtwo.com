Core Network APIs

Working Group Two's API platform allows both Mobile Operators and Third Party Developers to build products that interact with the core network. Operators can authenticate directly using API keys, while Third Party Developers need to obtain consent from subscribers via our Oauth2 service.

Third Party Developers

If you're a third party developer (someone who has no relation to us), you can can create an account in our Developer Portal at https://developer.wgtwo.com. From here you can create OAuth clients, which will allow our subscribers to grant you an access token to act on their behalf.

Mobile Operators

If you work for one of our partner operators, you can log in to our Partner Portal at https://console.wgtwo.com. From here you can create API keys which will grant you full access to all APIs for all of your subscribers. If you follow the examples you will be up and running within minutes!
